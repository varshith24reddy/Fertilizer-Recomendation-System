# Smart Fertilizer Recommendation System 

## Description
This project recommends the best fertilizer using Machine Learning based on soil and crop conditions.

## Technologies Used
- Python
- Flask
- Machine Learning (Decision Tree)

## How to Run
1. Install libraries:
   pip install flask pandas scikit-learn

2. Run model:
   python model.py

3. Run app:
   python app.py

4. Open in browser:
   http://127.0.0.1:5000/

## Output
The system predicts the best fertilizer based on user inputs.
